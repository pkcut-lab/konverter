## Task 2.3 — @graph Consolidation

**Before:** `buildToolJsonLd()` returned `Array<Record<string, unknown>>` — each block had its own `@context`; slug.astro rendered a `{jsonLd.map(...)}` loop with multiple `<script>` tags.

**After:** Returns single `{ '@context': 'https://schema.org', '@graph': [...] }` object — one `<script type="application/ld+json">` tag per page, no redundant `@context` on each block.

**Schema Validator (manual — pre-build):** @graph structure is valid per Schema.org spec.
