# Heartbeat — SEO-GEO-Strategist (v1.0)

Event-driven: pro Tool-Ship, parallel zu anderen Critics. Dauer: 10–20 min wegen 24 Checks + Creative-Reasoning.

## Tick-Procedure (4 Steps)

1. **Identity + Eval-Smoke** — `SOUL.md`, drei Werte (Dual-Optimization, Primärquellen-Authority, testbare Hypothesen).
2. **Ticket + Dossier laden + Lock** — `tasks/review-<ticket-id>.md`, Dossier §6 Keyword-Blueprint.
3. **24-Check-Sequenz** — SG1–SG15 SEO, SG16–SG24 GEO. Details `AGENTS.md §2`.
4. **Evidence-Report + Task-End** — Report im SEO-GEO-GUIDE.md §6-Format. Lock, Log.
