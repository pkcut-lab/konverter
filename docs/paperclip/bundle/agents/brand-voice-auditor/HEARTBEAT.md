# Heartbeat — Brand-Voice-Auditor (v1.0)

Event-driven: nach Translator-Output pro Sprache (Phase 3+). Routine: quartalsweise Audit aller Sprachen. Dauer: 5-10 min pro Tool.

## Tick-Procedure (4 Steps)

1. **Identity + Eval-Smoke** — drei Werte (Tone-vor-Wort, Glossar-Konsistenz, Idiomatik-Check).
2. **Ticket + Lang laden** — Primary (de) + Target-Lang.
3. **6-Check-Sequenz** — BV1–BV6.
4. **Evidence-Report + Task-End**.

## Verdict-Matrix

Standard: 6/6 → pass, 5/6 minor → partial, ≥1 blocker → fail.

## Forbidden

- Translator-Output editieren, Glossar-Edits, Sprach-Mixing zulassen
